Website resmi **Skanilan Tech**, sebuah *tech agency* yang dikelola oleh siswa **SMKN 9 Semarang** di bawah naungan **Thinkora Student Company**. Kami menyediakan berbagai solusi teknologi mutakhir, mulai dari Web Development, layanan IoT (Internet of Things), hingga digitalisasi bisnis.

🌍 **Live Demo:** [skanilantech.my.id](https://skanilantech.my.id)

---

## ✨ Fitur Utama

- **Katalog Layanan Digital:** Menampilkan jasa unggulan seperti Web Development, SEO Optimization, dan Gamification (Mini-games).
- **Admin Discount Control:** Fitur pengelolaan diskon dinamis yang memungkinkan admin mengatur harga promo dan hitung mundur (*countdown*) secara real-time.
- **WhatsApp Integration:** Tombol pemesanan yang otomatis mengarahkan ke WhatsApp dengan pesan teks yang sudah diformat rapi.
- **Optimasi SEO & Metadata:** Menggunakan Metadata API Next.js dan JSON-LD Schema untuk visibilitas maksimal di mesin pencari.
- **Desain Modern & Responsif:** Tampilan elegan yang dioptimasi untuk perangkat mobile dan desktop menggunakan Tailwind CSS.
- **Keamanan Data:** Integrasi Row Level Security (RLS) dari Supabase untuk melindungi data sensitif.

---

## 🛠️ Tech Stack

Proyek ini dibangun menggunakan standar industri untuk menjamin kualitas dan performa:

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Database & Auth:** [Supabase](https://supabase.com/)
- **Animation:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Image Hosting:** [ImageKit](https://imagekit.io/)
- **Deployment:** [Vercel](https://vercel.com)
