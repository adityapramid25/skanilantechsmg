**Skanilan Tech** adalah pusat inovasi digital dari siswa **SMK Negeri 9 Semarang**. Website ini dirancang dengan standar industri untuk melayani kebutuhan pengembangan perangkat lunak dan solusi teknologi modern. Kami mengutamakan performa, aksesibilitas, dan desain yang bersih.

- 🌐 **Live Website:** [skanilantechsmg.vercel.app](https://skanilantechsmg.vercel.app)
- 🚀 **Core Focus:** Web Development, IoT Solutions, Mobile Apps, & Photography.

## 💻 Teknologi yang Digunakan
- **Frontend/Backend:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS (Apple-style / Minimalist)
- **State & Data:** TypeScript & Supabase
- **SEO & Tracking:** Google Search Console Integrated
- **Infrastructure:** Vercel

## 🛠️ Fitur Utama Website
- **True Multipage Experience:** Navigasi bersih dengan URL `/about`, `/product`, dan `/contact`.
- **High SEO Visibility:** Meta tag dinamis dan Sitemap XML yang teroptimasi.
- **Responsive Interface:** Pengalaman pengguna yang mulus di perangkat mobile maupun desktop.
- **Fast Loading:** Optimasi rendering menggunakan teknologi terbaru dari Next.js.
